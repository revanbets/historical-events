WHEN YOU HAVE AN EXTENSION UPDATE:

1. EDIT SOURCE FILES
   └─ /Users/mac/Desktop/Claude research app/extension-working/chrome-extension/

2. UPDATE VERSION NUMBER
   └─ manifest.json → "version": "X.Y.Z"

3. BUILD ZIP
   cd extension-working && zip -r ../hdb-extension-vX.Y.Z.zip chrome-extension/

4. UPDATE RESEARCH COMPANION PAGE
   File: research-companion.html

   Find: <a href="https://historical-events-databse.netlify.app/hdb-extension-vX.X.X.zip"
   Replace with: new version number

   Optional: Update "How to Use" section if features changed
   (Look for ExtensionBanner component, then the <details> section)

5. COMMIT BOTH FILES
   git add research-companion.html hdb-extension-vX.Y.Z.zip
   git commit -m "Update extension to vX.Y.Z in Research Companion page"

6. VERIFY LIVE
   https://historical-events-databse.netlify.app/research-companion.html

PLACEHOLDER — Replace these with real images before publishing to App Store.

Required image files:
- icon.png          1024x1024px   App icon (used everywhere)
- splash.png        1284x2778px   Launch screen (dark navy #0F1223 background)
- adaptive-icon.png 1024x1024px   Android adaptive icon foreground
- favicon.png       48x48px       Web favicon (if using expo web)

Quick way to create them for testing:
1. Open any image editor (Preview on Mac works)
2. Create a 1024x1024 image with a dark navy (#0F1223) background
3. Add "HDB" text in blue (#60a5fa) in the center
4. Export as icon.png
5. For splash.png, use 1284x2778 with the same design

Or use an online tool: appicon.co, makeappicon.com

The app will show an error on startup if these files are missing.
